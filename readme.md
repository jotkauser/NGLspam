# NGLspam
A easy to use script that can spam ngl.link inboxes

# How to install?
- Install [node.js](https://nodejs.org/dist/v20.12.2/node-v20.12.2-x64.msi)
- Clone this repo 
- Open terminal
- Install packages (npm i)
- Run it with `node .`
